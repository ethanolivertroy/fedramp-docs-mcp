# Get Control Requirements

Get all FedRAMP requirements mapped to NIST control: "$ARGUMENTS"

Example controls: AC-2, SC-13, IA-5

Use the fedramp-docs MCP server's get_control_requirements tool to find all KSI items and FRMR requirements that reference this control. Show the requirement ID, title, theme, and description.
